# Get-hl

## Description

Get and save hitomi.la images.

## Usage

    Usage: get-hl <URL>

    URL: URL of gallery or reader of hitomi.la.
         e.g. https://hitomi.la/galleries/1245570.html
           or https://hitomi.la/reader/1245570.html

## Build

This is written in Go Language. So, just build it:

    # cd get-hl
    # go build
